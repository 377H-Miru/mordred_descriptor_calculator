# 記述子の詳細説明

このドキュメントは、`calculate_descriptors.py` スクリプトで計算される分子記述子について解説します。
計算される記述子は、大きく「**Mordred記述子**」と「**カスタム記述子**」の2種類に分かれます。

---

## 1. Mordred記述子

`mordred`ライブラリを用いて計算される、1800種類以上の標準的な分子記述子です。これらの記述子は、分子の多様な側面を数値化するために設計されています。

- **0D記述子 (Constitutional)**: 原子数、分子量、結合数など、分子の組成に基づく最も基本的な情報。
- **1D記述子 (Topological)**: 分子のトポロジー（原子の結合関係）に基づく指数。分子のサイズや分岐度などを表現します。
- **2D記述子 (Connectivity)**: 2次元の化学構造から計算される、より複雑な指数。環の数や電子状態に関する情報を含みます。
- **3D記述子 (Geometrical)**: 分子の3次元構造（立体配座）から計算される記述子。分子の形状や表面積などを含みます。

### PathCount記述子の拡張

本スクリプトでは、標準の`mordred`記述子に加え、**PathCount記述子**を**最大50次まで**拡張して計算しています。

- **MPC (Molecular Path Count)**: 指定された長さ（次数）のパス（原子の連なり）が分子内にいくつ存在するかを数えます。
- **piPC (pi Path Count)**: π電子系（二重結合や三重結合、芳香環）のみを考慮してパスを数えます。

これにより、標準設定では捉えきれない、中分子やポリマー様構造における長距離の構造的特徴をより詳細に表現できます。

### 詳細なリスト

全てのMordred記述子の詳細なリストと解説については、以下の公式ドキュメントをご参照ください。

- **Mordred公式ドキュメント:** [mordred-descriptor.github.io/documentation/master/](https://mordred-descriptor.github.io/documentation/master/)

## 2. カスタム記述子

分子内の**π共役系**に特化して、その物理化学的・構造的特徴を捉えるために独自に計算される記述子です。

### `Conjugation_Count`

- **目的**: 分子内に、互いに独立したπ共役系がいくつ存在するかを数えます。
- **解釈**: 値が大きいほど、分子が複数の分離した共役部位を持つことを示します。（例: ビフェニルでは1、スチレンとアセトフェノンが別の分子として存在するなら2）

### `Conjugation_MaxAtomCount`

- **目的**: 分子内で最も大きな共役系が、いくつの原子で構成されているかを測ります。
- **解釈**: 共役系の「広がり」や「サイズ」を示す指標です。値が大きいほど、広範囲に非局在化したπ電子系を持つことを示唆します。

### `Conjugation_MaxLength`

- **目的**: 最も大きな共役系内での、最も遠い原子間の経路長（グラフ理論における直径）を測ります。
- **解釈**: 共役系の「直線性」や「細長さ」を示す指標です。同じ原子数でも、環状構造より直鎖状構造の方が一般に大きい値を持ちます。

### `Conjugation_BLA` (Bond Length Alternation)

- **目的**: 最大共役系内の結合長の「ばらつき」を評価し、π電子の非局在化の度合いを測ります。
- **計算方法**: 3D座標に基づき、最大共役系に含まれる全ての結合長の**標準偏差**を計算します。
- **解釈**:
    - **値が0に近い**: 結合長が均一であることを意味し、π電子が高度に非局在化している状態を示唆します（例: 理想的なベンゼン環）。
    - **値が大きい**: 単結合と二重結合の性質が交互に強く現れている状態（結合長交互）を示唆します。
- **注意**: この記述子は3D構造が必須であり、座標がない場合は `NaN` (Not a Number) となります。

### `Conjugation_GraphEnergy`

- **目的**: Hückel法に類似したトポロジー的な考え方で、π共役系の電子的な安定性を評価します。
- **計算方法**: 最大共役系のトポロジー（原子の結合関係）から隣接行列を作成し、その固有値を求め、**固有値の絶対値の総和**を計算します。
- **解釈**: 一般に、この値が大きいほど、そのπ共役系がトポロジー的に安定であることを示唆します。
